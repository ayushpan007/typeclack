# Audio Attribution

The `media/blue/sound.ogg`, `media/brown/sound.ogg`, and
`media/red/sound.ogg` sound sprites were sourced from the MechvibesDX
pre-installed keyboard soundpacks:

- `cherrymx-blue-pbt`
- `cherrymx-brown-pbt`
- `cherrymx-red-abs`

Source repository: https://github.com/hainguyents13/mechvibes-dx

The MechvibesDX GitHub project page identifies the project license as MIT.
The per-key slice timings used by TypeClack come from each pack's
`config.json`.
